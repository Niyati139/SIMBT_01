
#   Tic-Tac-Toe game with Java
In this game, two players will be played and you have one print board on the screen where from 1 to 9 number will be displayed or you can say it box number. Now, you have to choose X or O for the specific box number. For example, if you have to select any number then for X or O will be shown on the print board, and turn for next will be there.The task is to create a Java program to implement a 3Ã—3 Tic-Tac-Toe game for two players.
